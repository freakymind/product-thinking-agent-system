## Overview

Agent PM is an **agentic AI system** that helps product teams create comprehensive Product Requirements Documents (PRDs) through conversations with specialized AI agents. Each agent is an expert in one domain—from user research to data modeling—and they collaborate autonomously to build structured requirements.

### Why Agent PM?

- **70% of failed projects** cite inadequate requirements as the root cause
- Traditional requirements processes skip critical thinking steps
- Agent PM guides you through rigorous product discovery with **10 specialized agents**

## The Agents

1. **ATLAS** - Project Context Gatherer
2. **IRIS** - User Signal Collector
3. **CLARA** - Problem Framer
4. **MAYA** - Jobs-to-be-Done Analyst
5. **VANCE** - Critical Validator
6. **NOVA** - Solution Explorer
7. **AXON** - Data Architect
8. **FLUX** - Feature Designer
9. **PRIME** - Priority Ranker
10. **SAGE** - AI Strategy Expert

## Privacy-First Architecture

Your requirements stay **100% private**:

- All data stored in **browser localStorage** only
- When you use your own API keys, requests go **directly from your browser to the LLM provider**
- We **never see, store, or access** your conversations or requirements
- [Read the full privacy architecture →](https://yoursite.com/privacy-architecture)

## Quick Start

1. Visit the app
2. Talk to ATLAS (the project context agent)
3. Navigate to other agents in any order
4. Export your PRD when ready

## Technology Stack

- **Frontend:** Next.js 16, React 19, TypeScript
- **AI:** Vercel AI SDK 6.0 (supports OpenAI, Anthropic, Google, AWS Bedrock)
- **Styling:** Tailwind CSS v4, shadcn/ui
- **State:** Browser localStorage (zero server state)
- **Architecture:** Agentic AI with autonomous context flow

## Development

